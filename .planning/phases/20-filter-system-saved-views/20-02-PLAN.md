```typescript
interface SavedViewsDropdownProps {
  currentFilter: TradeFilterState;
  onLoadView: (filter: TradeFilterState) => void;
  isGuest: boolean;
  initialViews: SavedView[];  // from settingsData parsed in TradesShell — avoids duplicate fetch
}
```

**Component behavior:**

1. **State:** `views: SavedView[]` initialized from `initialViews` prop. `showDropdown: boolean`, `showNameInput: boolean`, `viewName: string`.

2. **Sync initialViews:** Use a useEffect to sync when `initialViews` changes (e.g., on page load when settingsData arrives):
   ```typescript
   useEffect(() => { setViews(initialViews); }, [initialViews]);
   ```

3. **Render:** A button labeled "Views" (Bookmark icon from lucide-react) with a count badge showing `views.length` if > 0. Clicking opens a dropdown.

4. **Dropdown contents:**
   - List of saved views, each showing: view name, created date (formatted short), a load button, and a delete button (Trash2 icon).
   - Clicking load: calls `onLoadView(view.filter)` and closes dropdown.
   - Clicking delete: calls `deleteView(view.id)`.
   - At the bottom: "Save current filters" button that shows a name input + "Save" button.
   - If no views exist, show "No saved views" placeholder text.

5. **Save view (FILT-08):**
   ```typescript
   const saveView = async (name: string) => {
     const view: SavedView = {
       id: crypto.randomUUID(),
       name,
       filter: currentFilter,
       created_at: new Date().toISOString(),
     };
     const updated = [...views, view];
     setViews(updated);
     setShowNameInput(false);
     setViewName("");
     if (!isGuest) {
       await fetch("/api/settings", {
         method: "PUT",
         headers: { "Content-Type": "application/json" },
         body: JSON.stringify({ saved_filter_views: JSON.stringify(updated) }),
       });
     }
   };
   ```

6. **Delete view (FILT-09):**
   ```typescript
   const deleteView = async (id: string) => {
     const updated = views.filter(v => v.id !== id);
     setViews(updated);
     if (!isGuest) {
       await fetch("/api/settings", {
         method: "PUT",
         headers: { "Content-Type": "application/json" },
         body: JSON.stringify({ saved_filter_views: JSON.stringify(updated) }),
       });
     }
   };
   ```

7. **Outside click to close:** Use useRef on the dropdown container and useEffect with document mousedown listener (same pattern as column menu in TradesShell).

8. **Styling:** Match existing dropdown patterns — `absolute right-0 top-full mt-2 z-50 w-64 rounded-lg dark:bg-slate-800 bg-white shadow-xl py-2`. View items use `hover:dark:bg-slate-700/50 hover:bg-slate-50`. Delete button is `text-red-400 hover:text-red-300`. Save button uses emerald theme.

9. **Guest behavior:** Guests can still use views in-memory (they work for the current session) but they are NOT persisted to /api/settings. The `if (!isGuest)` guard handles this.
