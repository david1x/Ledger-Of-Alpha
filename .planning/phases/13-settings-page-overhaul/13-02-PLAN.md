    Each tab component already fetches settings on mount and has local state. Add dirty-state tracking:

    1. Accept `onDirtyChange?: (dirty: boolean) => void` as a prop
    2. Store a `savedBaseline` ref (the values as they were when last fetched or saved)
    3. Compare current state to baseline on each state change
    4. Call `onDirtyChange(true)` when any field differs from baseline
    5. Call `onDirtyChange(false)` after a successful save (which also updates the baseline)

    Pattern for each tab (example for AccountTab):
    ```typescript

    }

      const [accountSize, setAccountSize] = useState("10000");
      // ... other state
      const baselineRef = useRef<Record<string, string>>({});

      useEffect(() => {
        fetch("/api/settings").then(r => r.json()).then(data => {

            risk_per_trade: data.risk_per_trade ?? "1",
            // ... other fields
          };
          setAccountSize(vals.account_size);
          // ... set other state
          baselineRef.current = vals;
        });
      }, []);

      // Dirty check
      useEffect(() => {
        const current = { account_size: accountSize, risk_per_trade: riskPerTrade, /* ... */ };

        onDirtyChange?.(dirty);
      }, [accountSize, riskPerTrade, /* ... other deps */]);

      const handleSave = async () => {
        // ... existing save logic
    ```

    In `app/settings/page.tsx`:

       ```typescript
       const handleDirtyChange = useCallback((tabId: Category) => (dirty: boolean) => {

           else next.delete(tabId);

         });
       ```
