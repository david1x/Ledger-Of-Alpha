**lib/calculators.ts** — Create with ALL six calculator math functions (pure, no React):

1. `drawdownRecovery(drawdownPct: number): number` — Returns required gain % to recover. Formula: `((1 / (1 - d)) - 1) * 100` where d = drawdownPct/100. Return 0 if drawdownPct <= 0, Infinity if >= 100.

2. `kellyFraction(winRatePct: number, avgWin: number, avgLoss: number): number` — Returns optimal position size %. Formula: `(b * p - q) / b * 100` where b = avgWin/avgLoss, p = winRatePct/100, q = 1-p. Clamp to [0, 100]. Return 0 if any input <= 0 or winRate >= 100.

3. `fibonacciLevels(high: number, low: number): { label: string; price: number; isExtension: boolean }[]` — Retracement levels: 0%, 23.6%, 38.2%, 50%, 61.8%, 78.6%, 100%. Extensions: 127.2%, 161.8%, 261.8%. Price = high - range*ratio for retracements; low - range*(ratio-1) for extensions.

4. `compoundGrowthCurve(startBalance: number, monthlyReturnPct: number, months: number): { month: number; balance: number }[]` — Array of month/balance pairs. balance = startBalance * (1 + rate)^i.

5. `riskReward(entry: number, stopLoss: number, takeProfit: number, direction: "long" | "short"): { riskPerShare: number; rewardPerShare: number; rrRatio: number; riskPct: number; rewardPct: number }` — Extract R:R math from RiskCalculator.tsx pattern. riskPerShare = abs(entry - stop), rewardPerShare = abs(target - entry). For long: stop < entry < target. For short: target < entry < stop.

6. `positionSize(accountSize: number, riskPercent: number, entry: number, stopLoss: number, commission?: number): { dollarRisk: number; shares: number; positionValue: number }` — Extract from PositionSizer.tsx pattern. shares = floor(adjustedRisk / stopDistance).
