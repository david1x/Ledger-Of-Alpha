  <files>components/TradeModal.tsx</files>
  <action>
Modify TradeModal.tsx to add MonteCarloPreview below PositionSizer in the Setup tab.

**New state variables:**
- `mcTrades`: array of `{ pnl_percent: number; strategy_id?: string | null }[]` -- historical closed trades for the selected account
- `mcCollapsed`: boolean, loaded from settings API key `montecarlo_panel_collapsed` (default true -- collapsed by default per user decision)
- `mcRuinThreshold`: number, loaded from settings API key `montecarlo_ruin_threshold` (default 5)

**Data fetching:**
- In the existing useEffect that loads account data (or a new one), fetch closed trades for `selectedAccountId`:
  ```
  GET /api/trades?account_id={selectedAccountId}&status=closed
  ```
- Map response to `{ pnl_percent, strategy_id }` objects. Compute `pnl_percent` as:
  - If trade has `pnl_percent` field, use it directly
  - Otherwise compute from `(exit_price - entry_price) / entry_price * 100` (adjusted for direction)
- Re-fetch when `selectedAccountId` changes

**Settings loading:**
- On mount, fetch `montecarlo_panel_collapsed` and `montecarlo_ruin_threshold` from `/api/settings`
- These may already be fetched if TradeModal loads settings -- check existing pattern and reuse

**Settings persistence:**
- When MonteCarloPreview's `onCollapsedChange` fires, save to `/api/settings` with key `montecarlo_panel_collapsed`
- Debounce or fire-and-forget the save (non-critical)

**Render MonteCarloPreview** immediately after the `</PositionSizer>` closing tag (line ~406), inside the same `<div className="space-y-8 ...">`:

```tsx
<MonteCarloPreview
  historicalTrades={mcTrades}
  strategies={strategies.map(s => ({ id: s.id, name: s.name }))}
  startingBalance={form.account_size ?? accountSize}
  entry={form.entry_price ?? null}
  stopLoss={form.stop_loss ?? null}
  direction={form.direction ?? "long"}
  currentShares={form.shares ?? null}
  ruinThreshold={mcRuinThreshold}
  commission={form.commission ?? defaultCommission}
  onApplyShares={(s) => set("shares", s)}
  onCollapsedChange={(c) => { setMcCollapsed(c); saveSetting("montecarlo_panel_collapsed", c); }}
  defaultCollapsed={mcCollapsed}
/>
```

**Important:** Use `selectedAccountId` (the trade's account) for balance lookup, NOT `activeAccountId`, per architecture decision. The `accountSize` state already reflects this since it's set from the selected account.

**Import:** Add `import MonteCarloPreview from "./MonteCarloPreview";` at top.
