Create a small reusable "use client" component `ChecklistRing` that renders a circular SVG progress ring (donut chart).

**Props:**
- `checklistState: string | null | undefined` — raw JSON string from trade.checklist_state
- `size?: number` — diameter in pixels (default 28)

**Behavior:**
- Parse the JSON string into `ChecklistItem[]`. If null, empty, or invalid JSON, return `null` (render nothing — per user decision: no badge if no checklist data).
- Count `checked` items vs `total` items.
- Render a small SVG circle with:
  - Background circle: `stroke` in muted color (dark:slate-700 / slate-200)
  - Foreground arc: `stroke` in emerald, `strokeDasharray` + `strokeDashoffset` to show the fill ratio
  - Center text: `"X/Y"` in tiny font (e.g., 8px bold)
- Use `strokeLinecap="round"` for a clean arc appearance.
- If all items checked (100%), use a brighter emerald or green color.
- If 0 items checked but items exist, show empty ring with "0/Y".

**Styling:** Dark mode first per project conventions. Use Tailwind for the text, raw SVG attributes for the ring.

Export as default: `export default function ChecklistRing(...)`.
